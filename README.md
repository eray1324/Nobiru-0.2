# NOBIRU - Plataforma Educativa

**NOBIRU** significa "crecer y mejorar constantemente" en japonés.

Una plataforma educativa moderna diseñada para estudiantes, con características de aprendizaje colaborativo, cuestionarios interactivos, reels educativos y una biblioteca de recursos.

## ✨ Características

- 📝 **Cuestionarios**: Crea y resuelve cuestionarios personalizados
- 💬 **Comunidad**: Crea preguntas, comparte consejos y aprende con otros
- 🎬 **Reels Educativos**: Videos educativos por categoría
- 📚 **Biblioteca**: Almacena y comparte PDFs, libros y diapositivas
- ⭐ **Favoritos**: Guarda tus recursos favoritos
- 🏆 **Sistema de Insignias**: 10 niveles de logros diferentes
- 🌙 **Modo Oscuro**: Diseño elegante y amigable con los ojos
- 📱 **Responsive**: Funciona perfectamente en móvil y PC

## 🛠️ Tecnologías

- **Backend**: Python, Flask, Flask-SQLAlchemy
- **Frontend**: HTML, CSS, JavaScript Vanilla
- **Base de Datos**: SQLite
- **Deploy**: Render
- **Versión**: GitHub

## 📋 Requisitos

- Python 3.8+
- pip (gestor de paquetes de Python)
